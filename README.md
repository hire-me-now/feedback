# hire-me.now — feedback

The place to **report bugs, request features, and ask questions** about
[hire-me.now](https://resume.hire-me.now). The product code is private; this repo is
open so anyone can weigh in.

## How to reach us

- 🐞 **Bug?** [Open an issue](https://github.com/hire-me-now/feedback/issues/new/choose) → *Bug report*.
- 💡 **Idea or feature?** [Open an issue](https://github.com/hire-me-now/feedback/issues/new/choose) → *Feature request*.
- 💬 **Question or general feedback?** Use [Discussions](https://github.com/hire-me-now/feedback/discussions).

## Before you post

- **Search first** — it may already be here.
- **This repo is public** — don't paste your API key or personal data you'd rather not share.
- The more specific (steps, browser, a screenshot, console errors), the faster it's sorted.

Thanks for helping make it better. 🙏
